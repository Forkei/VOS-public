Table 1. VOS agent hierarchy showing each agent's role and available tools. All sub-agents share a common base of standard tools for inter-agent messaging, task management, and memory operations.
Agent Role Domain-Specific Tools
Primary Agent Central orchestrator and sole user interface. Analyzes intent, delegates to sub-agents, synthesizes responses. Messaging: send_user_message, send_agent_message Tasks: create_task, update_task, get_tasks, assign_to_task Memory: create_memory, search_memory, update_memory, delete_memory
Weather Agent Provides current conditions, forecasts, UV index, and air quality data via OpenWeatherMap API. get_weather, get_forecast, get_weather_by_coordinates, get_uv_index, get_air_quality
Notes Agent Manages persistent notes with Weaviate semantic search integration for intelligent retrieval. create_note, list_notes, get_note, update_note, delete_note, search_notes, archive_note, pin_note
Calendar Agent Handles scheduling with calendar events and time-based reminders. Events: create_calendar_event, list_calendar_events, update_calendar_event, delete_calendar_event Reminders: create_reminder, list_reminders, edit_reminder, delete_reminder
Search Agent Web research via Firecrawl (scraping, crawling, extraction) and DuckDuckGo (text, news, books). Firecrawl: firecrawl_scrape, firecrawl_crawl, firecrawl_map, firecrawl_search, firecrawl_extract DuckDuckGo: ddg_text_search, ddg_news_search, ddg_books_search
Browser Agent AI-powered browser automation for web navigation, form filling, and data extraction via natural language. browser_use, browser_navigate
Calculator Agent Mathematical computation including arithmetic, statistics, linear algebra, and unit conversion. basic_calculation, advanced_math, statistics, random_number, number_theory, linear_algebra, unit_conversion
All sub-agents inherit standard tools: send_agent_message (reply to Primary), task management (create/update/get tasks), memory operations (CRUD on Weaviate), and lifecycle control (sleep, shutdown).
